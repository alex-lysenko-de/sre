## Публичные vs защищённые

- `meta.public: true` — доступны без регистрации вообще (гостевой доступ,
  см. [[Модель-аутентификации]]): `Login`, `Welcome`.
- `requiresAuth: false` без `public` — `Main`, `Info`: доступны и гостям, и
  авторизованным (не редиректят на `/login`), но не входят в «гостевой
  белый список» наравне с `public`.
- `requiresAuth: true` — обязательна активная сессия + активный пользователь
  в [[users]].
- `requiresAdmin: true` — дополнительно `role === 'admin'`.

## `?id=`/`?n=` → `/armband/:id`

В конце `router.beforeEach` — если у **любого** маршрута в query есть `id`
или `n`, происходит редирект на `{ name: 'Armband', params: { id } }`. Это
и есть точка входа публичного QR-URL (см. [[QR-код-URL]]) — само приложение
открывается по короткой ссылке с параметром, а не по прямому пути
`/armband/:id`.
