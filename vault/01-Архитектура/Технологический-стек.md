# Технологический стек

| Компонент | Описание |
|---|---|
| Framework | Vue.js 3 (в коде — смесь Composition API `<script setup>` и Options API, см. [[Структура-каталогов]]) |
| Styling | Bootstrap 5.3.3 + Font Awesome (иконки регистрируются централизованно в `src/main.js` через `library.add(...)` перед использованием в шаблонах) |
| State management | Pinia — `stores/user.js`, `stores/config.js`; используется в новом коде, миграция остального кода на Pinia — в процессе (см. `readme.md`, «ARCHITECTURE & DESIGN») |
| Backend | Supabase (Postgres + Auth + Edge Functions), см. [[Обзор-схемы-БД]] и [[Обзор-Edge-Functions]] |
| Локальное кэширование | LocalForage (IndexedDB), см. [[Кэширование-LocalForage]] |
| App-Typ | Progressive Web App (PWA) |
| Build Tool | Vite |
| Package Manager | npm (Node.js 20+) |
| Hosting | GitHub Pages, см. [[Сборка-и-деплой]] |

## Требования к стилю кода (из `readme.md`)

- Язык UI — немецкий (Deutsch)
- Язык комментариев/логов/debug-сообщений — немецкий или английский
- **Никаких русских слов** в коде, комментариях или тексте UI (в отличие от
  документации/vault, где принят русский — см. `_Конвенции.md`)
- Кодировка UTF-8, отступы — 2 пробела
- ESLint + Prettier
- Импорты — через алиас `@`
- Секреты/credentials никогда не хардкодятся
- Все запросы к БД изолируются в отдельных модулях `src/composables/useXXX.js`
  — минимизация связанности с конкретной БД (см. [[Обзор-composables]] и
  [[Целевая-архитектура]])
