# `children`

> Источник: `doc/table_structure.md`.

Справочник детей — статичные данные, не привязанные к конкретному дню.

```sql
create table public.children (
  id bigint generated by default as identity not null,
  created_at timestamp without time zone not null,
  name character varying null,
  age smallint null,
  schwimmer smallint null default '0'::smallint,
  status smallint null default '0'::smallint,
  notes text null default '""'::text,
  group_id bigint null,
  band_id bigint null,
  constraint children_pkey primary key (id),
  constraint children_band_id_key unique (band_id)
);
```

- `band_id` — уникальный идентификатор браслета (`UNIQUE`), может быть
  переназначен на другого ребёнка (замена браслета, см.
  [[Идентификация-ребёнка]]).
- `group_id` — группа ребёнка (число, не FK на отдельную таблицу групп — см.
  [[Обзор-схемы-БД]]).
- `schwimmer` — признак «умеет плавать» (используется в
  `BusDetailModal.vue`/`fetchBusChildren()`).
